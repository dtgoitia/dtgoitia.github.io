AutoCAD C# 'HelloWorld'

October 23, 2016

Are you a newbie looking for a start point to get started in C# for AutoCAD? Take a look to this post then. As mentioned in my previous post, I did my first C# written HelloWorld within AutoCAD 2017 with Visual Studio 2015. Here what I did:

STEP 0: prerequisites

For this process I have installed the following:

  • AutoCAD 2017
  • Visual Studio 2015
  • AutoCAD ObjectARX SDK

Now on I will assume you have already downloaded and installed them.

STEP 1: create a new project

On Start Page tab, select New Project…

Select the template for an AutoCAD C# project at: Installed > Templates > Visual C# > Autodesk

Note you will have some default values at Name and Solution name fields (bottom of the window). Override the Name field with any name you want to give to this project, any name is valid. Solution name field will fill up automatically too when you write on Name field. Hit OK.

As shown below, we need to tell to Visual Studio where to find AcMgd.dll library and acad.exe executable:

  • AcMgd.dll: it is wherever you have downloaded the ObjectARX SDK.
  • acad.exe: if you have use default installation directory, it should be at C:\Program Files\Autodesk\AutoCAD 2017.

Hit OK to create your project.

STEP 2: insert our code

At the Solution Explorer tab, find myCommand.cs file and click on it.
